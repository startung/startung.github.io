# startung.nl

Personal website project for startung.nl.

## Features
- **Static Site**: Built with semantic HTML5, CSS3, and Vanilla JavaScript.
- **Theme Switcher**: Dark and Light mode support, persisted via `localStorage`.
- **Responsive Design**: Mobile-friendly layout and navigation.
- **Contact Form**: Functional form powered by Formspree.

## Pages & Structure
- **`index.html`**: The main landing page (About Me).
- **`publications.html`**: A list of academic publications including:
    - Summaries and direct links to papers (CEUR-WS).
    - Keyword tags for quick topic identification.
- **`blog.html`**: A consolidated blog section for sharing thoughts.
- **`ai_glossary.html`**: A dedicated page for AI terminology.
- **`contact.html`**: specific contact form for getting in touch.

## Technical Details
- **CSS**: Located in `css/style.css`. Uses CSS variables for theming and consistent styling.
- **JavaScript**: Located in `js/script.js`. Handles the theme toggle logic.
- **Assets**: Images and other static assets are organized in their respective directories.

## Development
- `log.md`: Keeps track of development progress and feature implementations.
